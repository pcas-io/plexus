# Weekly Review

## 1. Inbox leeren
- Alle ungeplanten Tasks durchgehen (Backlog ohne Projekt)
- Für jeden: Zuordnen, Terminieren, Delegieren oder Löschen

## 2. Projekte prüfen
- Jedes aktive Projekt: Gibt es einen nächsten Schritt?
- Blockierte Tasks: Was fehlt?
- Überfällige Tasks: Neues Datum oder deprioritisieren?

## 3. Offene Actions
- Recurring Actions: Sind alle aktuell?
- Neue Actions nötig?

## 4. Incidents
- Offene Incidents: Status-Update nötig?
- Resolved Incidents: Post-Mortem geschrieben?

## 5. Nächste Woche
- Top 3 Prioritäten festlegen
- Meilensteine diese Woche?
- Blocker identifizieren

## Tools
- `context_load` → Überblick
- `list_tasks status:backlog` → Inbox
- `list_actions` → Offene Actions
- `list_incidents` → Offene Incidents
